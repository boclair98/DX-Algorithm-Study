# DX-Algorithm-Study

알고리즘 문제 풀이를 기록하는 학습 저장소입니다.

백준 문제를 중심으로 풀이 코드를 정리하며, 구현력과 기본 자료구조/알고리즘 감각을 꾸준히 유지하기 위해 사용합니다.

## 풀이 목록

| 플랫폼 | 문제 | 주제 |
| --- | --- | --- |
| Baekjoon | 2309 일곱 난쟁이 | 브루트포스 |
| Baekjoon | 25305 커트라인 | 정렬 |
| Baekjoon | 2587 대표값2 | 정렬, 평균/중앙값 |
| Baekjoon | 4150 피보나치 수 | 큰 수 처리 |

## 저장 방식

```text
문제_폴더/
└── 1.py
```

각 문제별 폴더에 풀이 코드를 저장합니다.

## 학습 목표

- 문제를 작은 조건으로 분해하는 연습
- 반복문, 조건문, 정렬, 리스트 처리 기본기 강화
- 풀이 기록을 남겨 다시 복습 가능한 형태로 관리

## 코드 등록 규칙

1. 풀이한 알고리즘 문제 폴더로 이동합니다.
2. 새 풀이 파일을 생성합니다.
3. 파일명은 순서에 맞는 숫자와 `.py` 확장자를 사용합니다.

예시:

```text
백준_2309번_(일곱난쟁이)/1.py
```
